------------- STEPS TO RUN PROJECT --------------------

1. Extract file into C:/Xampp/Htdocs/Grade/   (create a folder name "Grade" in Htdocs folder).
2. Open localhost/phpmyadmin
3. Click on Database tab and create a database named "gdb" 
4. click on import tab and import "gdb.sql" file from database folder from Grade folder.
5. open chrome and type url-localhost/grade/home.html
